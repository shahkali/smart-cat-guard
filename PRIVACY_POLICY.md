# Privacy Policy for Smart Cat Guard

Effective date: May 7, 2026

Smart Cat Guard ("the extension") is a productivity Chrome extension that helps users enforce healthy browsing breaks. This Privacy Policy explains what data the extension handles.

## Data Collection

Smart Cat Guard does **not** collect personally identifiable information.

The extension does **not** collect or transmit:
- name, email, phone number, or address
- health, financial, or authentication data
- personal communications
- precise location data
- website page content (text, images, audio, video)
- clickstream, keystrokes, or behavioral tracking analytics

## Local Data Storage

To provide its core functionality, Smart Cat Guard stores settings locally in the browser using Chrome's `storage` API, including:
- usage limit (minutes)
- break duration (minutes)
- enabled tracked default sites
- user-added custom site domains
- active break state and break expiry timestamp

This data is stored locally on the user's browser profile and is not sold or shared.

## Permissions Use

Smart Cat Guard uses Chrome permissions only for its single purpose (timed break enforcement):

- `storage`: save and load user settings and break state
- `tabs`: detect tracked tabs and synchronize break overlay behavior
- `alarms`: run reliable timers for usage and break durations
- host permission `<all_urls>`: allow the extension to run on user-selected tracked domains, including custom websites added by the user

## Remote Code and Third Parties

- The extension does not use remote code.
- The extension does not load external JavaScript libraries from CDNs.
- The extension does not sell or transfer user data to third parties.

## Children's Privacy

Smart Cat Guard is not directed to children under 13 and does not knowingly collect personal data from children.
